# Beach Niceness Forecast

A beach weather index for Portuguese beaches, combining temperature and wind data to calculate a niceness score for the next 24 hours.

Built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui. Weather data from [Open-Meteo](https://open-meteo.com/).

Deployed to GitHub Pages: https://ricardosou.github.io/beach-niceness-forecast/

## Development

```sh
npm install
npm run dev
```

## Deploy

Pushes to `main` automatically build and deploy via GitHub Actions.
